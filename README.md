# file-organizer
.An automated file organizer built with Python that sorts files into categorized folders such as images, videos, and documents for better file management.
# 📂 File Organizer

## 📌 Description

A Python automation script that organizes files into folders based on their file types. It helps keep directories clean and structured automatically.

---

## 🚀 Features

* Automatically sorts files into folders
* Supports images, videos, documents, and more
* Easy to use and customizable
* Saves time by automating manual file management

---

## 🛠️ Technologies Used

* Python
* `os` module
* `shutil` module

---

## 📁 How It Works

* Scans all files in a selected directory
* Detects file types using extensions
* Moves files into respective folders like:

  * Images
  * Videos
  * Documents

---

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/Satnamsingh144/file-organizer.git
```

2. Navigate to the project folder:

```bash
cd file-organizer
```

3. Run the script:

```bash
python main.py
```
